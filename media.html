Live media sync

Медиа Royce Family теперь подтягивает аватары и статистику автоматически

Теперь можно вообще не лезть в код страницы. Добавляй ссылки через админку или в media-sources.json, а сайт сам берёт аватарки и счётчики с YouTube и Twitch через backend-роут /api/media/resolve.

Открыть админку медиа
Как теперь удобно

Редактирование ссылок без ручной правки html

Если в браузере сохранены свои медиа-источники из админки — страница использует их. Если нет, автоматически берёт данные из файла media-sources.json.

Что делать теперь

1. Открой admin.html

В админке появился блок управления медиа, где можно добавить YouTube/Twitch ссылки через форму.

2. Сохрани карточку

Сайт запишет источник в localStorage и media-страница сразу начнёт использовать эти данные.

3. Нажми «Обновить сейчас»

Страница отправит список ссылок на backend и вернёт живые аватары/цифры.

Источник

admin.html

Самый быстрый путь: добавлять медиа не через код, а через визуальную форму.

Fallback

media-sources.json

Если локальных данных нет, подгружается стандартный JSON-файл проекта.

Кэш

Server cache

Сервер кэширует ответы, чтобы квота API не сгорала на каждом обновлении страницы.
